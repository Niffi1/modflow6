This is the steady-state stream-aquifer interaction example problem (test 1) for the parent model grid from U.S. Geological Survey Techniques and Methods 6-A34 (Mehl and Hill, 2010).  This problem is simulated using the Streamflow Routing (SFR) Package in MODFLOW 6. Mehl, S.W. and Hill, M.C., 2010, MODFLOW-LGR -- Modifications to the Streamflow-Routing Package (SFR2) to route streamflow through locally refined grids: U.S. Geological Survey Techniques and Methods 6-A34, 15 p.
